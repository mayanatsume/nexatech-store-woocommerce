# NexaTech Store

NexaTech Store is a responsive technology e-commerce project developed with WordPress and WooCommerce.

## Features

- Custom WordPress plugin
- WooCommerce product catalogue
- Product categories
- Shopping cart
- Checkout flow
- Custom payment interface
- Responsive navigation and mobile menu
- Product slider
- Custom homepage and WooCommerce templates
- Responsive design

## Technologies

- WordPress
- WooCommerce
- PHP
- JavaScript
- CSS
- Tailwind CSS structure
